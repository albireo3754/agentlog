# Review Memory

This directory separates LLM review memory from deterministic gates.

| File | Purpose |
| --- | --- |
| `SELF_REVIEW.md` | Curated context for LLM self-review. Read this before reviewing a diff. |
| `review-list.json` | Static gate rules promoted from recurring review findings. |
| `history.jsonl` | Local run log generated by `self-review`; ignored by git. |

## Static Gate

Run before pushing PR updates:

```bash
bun run self-review -- --mode staged
bun run typecheck
bun test
```

Useful modes:

```bash
bun run self-review -- --mode branch --base origin/develop
bun run self-review -- --mode staged
```

The script appends every run to `docs/review/history.jsonl`.
It fails when an applicable rule lacks required evidence.

## LLM Review

Use `SELF_REVIEW.md` as the review memory:

```text
Use docs/review/SELF_REVIEW.md as review memory.
Review the current diff against that memory.
Find new material issues not covered by static gates.
For each finding: severity, file:line, why, concrete fix, and whether it should become a static gate.
```

Promote repeatable findings into `review-list.json`.
